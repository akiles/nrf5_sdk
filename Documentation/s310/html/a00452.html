Detailed Description

GPIO pin abstraction and port abstraction for reading and writing byte-wise to GPIO ports.

Here, the GPIO ports are defined as follows:

  • Port 0 -> pin 0-7
  • Port 1 -> pin 8-15
  • Port 2 -> pin 16-23
  • Port 3 -> pin 24-31
